# 九州特种兵旅行 Review (2026/3/25 - 3/28)

## 总体评价

4 天 3 晚，逆时针环线（福冈→佐世保→平户→长崎→佐贺→唐津→太宰府→福冈），规划极其精细。
核心成就：日本本土最西端、大明海商故里、核爆原点、大明国宝古刹、绝景海城+夜樱、九州第一赏樱神社。

**评分: A+**

---

## JR Pass 结论：不买！

北九州 JR Pass 3 日券 ¥15,000，全程 JR 单买仅 ~¥12,490。无论怎么激活都亏 ¥2,500+。
建议用九州ネットきっぷ等网络折扣票进一步省钱。

## 修正建议总结

1. **不买 JR Pass**: 单买 ¥12,490 < Pass ¥15,000，用ネットきっぷ更省。
2. **Day 2 巴士确认**: 佐々→平户 10:06 发车，平户→佐世保 16:30 发车。
3. **Day 3 夜樱加餐**: 16:46 到唐津后直奔舞鶴公園看夜樱（日落~22:00）。
